# NN_L_04  
## 6.1 XOR	X1, X2	1 слой (3)	ReLU	-	Классическое решение XOR через скрытый слой  

<img width="2756" height="1375" alt="image" src="https://github.com/user-attachments/assets/c0e815bc-5300-452d-b9b5-eec828ef32c9" />  
11 эпох, Test loss 0.015, Training loss 0.016  

Сеть успешно решила задачу XOR благодаря наличию скрытого слоя с 3 нейронами и функции активации ReLU, что позволило создать нелинейную разделяющую поверхность (низкий loss 0.015-0.016). Без скрытого слоя эта задача была бы неразрешима, так как XOR линейно неразделим.  

## 6.2 Круги	Все 7 признаков	0 слоев	Sigmoid	-	Модель без слоев на избыточных данных  

<img width="2825" height="1375" alt="image" src="https://github.com/user-attachments/assets/b1130dc9-6e46-4eec-9804-91d3df82b1da" />  
13 эпох, Test loss 0.033, Training loss 0.026  

Несмотря на отсутствие скрытых слоёв, сеть смогла построить круговую границу решений благодаря использованию всех 7 признаков, включая полиномиальные (X1², X2²). Это демонстрирует, что добавление нелинейных признаков позволяет модели без скрытых слоёв решать нелинейные задачи.  

## 6.3 Спираль (Шум 10%)	X1, X2	2 слоя (8, 6)	Tanh	Reg: L2 (0.1)	"Сглаживание" границ регуляризацией  

<img width="2707" height="1378" alt="image" src="https://github.com/user-attachments/assets/f8d9e019-2524-4a12-b712-4fa7a3050f46" />  
106 эпох, Test loss 0.5, Training loss 0.5  

Сеть не смогла эффективно обучиться на спиральных данных (loss 0.500), так как L2-регуляризация с коэффициентом 0.1 оказалась слишком сильной для этой сложной задачи. Регуляризация чрезмерно "сглаживает" веса и ограничивает способность сети выучить сложную спиральную границу, что привело к отсутствию прогресса в обучении.  

При этом, даже с отключеной регуляризацией сеть все равно не может эффективно обучиться. Попробовав создать несколько других вариантов моделей с данными формата "спираль", можно сделать вывод что подобрать модель которая сможет обучиться на таких данных в целом проблематично
